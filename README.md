 Content Performance & Engagement Analysis
 Project Overview
 This is an end-to-end data analytics project focused on analyzing **content performance and        engagement metrics** across multiple social media platforms using **SQL and Power BI**.

The goal of this project is to identify:
- Which platforms drive the highest engagement
- Which content types perform best
- How engagement trends change over time
- Key KPIs useful for marketing and growth teams

Tools & Technologies
- SQL – Data querying, aggregation, KPI calculations
- Power BI – Data modeling, DAX measures, dashboard creation
- Excel / CSV – Source dataset
- GitHub– Project version control and documentation

  Key KPIs Calculated
- Total Views
- Total Engagement
- Engagement Rate
- Total Posts
- Engagement by Platform
- Engagement by Content Type
- Engagement Trend Over Time

 SQL Analysis Highlights
Using SQL, the following analyses were performed:
- Aggregation of views, likes, comments, and shares
- Platform-wise and content-type-wise engagement breakdown
- Engagement rate calculation
- Time-based engagement trend analysis

All SQL queries are available in the `02_SQL` folder.

 Key Insights
- Instagram generates the highest engagement compared to LinkedIn and Twitter
- Reels outperform other content types in terms of engagement
- Engagement shows noticeable variation over time, indicating campaign or posting strategy impact
- Visual KPIs help quickly identify performance gaps and growth opportunities

Business Use Case
This dashboard can be used by:
- Marketing teams to optimize content strategy
- Social media managers to identify high-performing platforms
- Analysts to track engagement trends and KPIs over time

 Conclusion
This project demonstrates the ability to:
- Perform structured SQL analysis
